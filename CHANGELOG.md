# 更新日志

所有重要的项目变更都会记录在此文件中。

## [1.0.0] - 2026-02-23

### 🎉 首次发布

这是"不知道叫什么名字小手机"的首个正式版本！

#### ✨ 核心功能

- **聊天系统**
  - 完整的多角色对话功能
  - 支持单聊和群聊
  - 消息格式化与编辑
  - 隐藏消息管理（节省Token）
  - 消息通知系统

- **AI集成**
  - API设置与管理
  - 支持多种AI模型
  - API调用历史记录
  - 智能格式调节

- **外观定制**
  - 多种主题切换
  - 自定义字体设置
  - 气泡样式自定义
  - 颜色方案管理
  - 锁屏界面（支持滑动/手势解锁）

- **数据管理**
  - 完整的数据导入导出
  - 浏览器存储容量监控
  - 图片压缩功能
  - 数据统计与可视化
  - 冗余数据清理

- **角色系统**
  - 世界书管理
  - 人设自定义
  - 角色关系设置
  - SillyTavern导入支持

- **小组件**
  - 情侣头像展示
  - 音乐搜索与播放
  - 视频通话界面
  - 个性化文案编辑

- **PWA支持**
  - 可安装到桌面
  - 离线使用
  - 原生应用体验

#### 🔧 技术特性

- 基于IndexedDB的本地存储
- 响应式设计，适配移动端
- iOS风格UI设计
- Service Worker支持
- 图片自动压缩

## 版本说明

版本号格式：`主版本号.次版本号.修订号`

- **主版本号**：重大功能更新或不兼容的API变更
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
